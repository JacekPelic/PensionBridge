## Pillar / hub structure

Three evergreen pillars, each linking to the spoke pages above. These are long (2000+ words), authoritative, build domain authority.

1. **`/guides/cross-border-pension-basics`** — the EU pension coordination primer. Links to every corridor page (Tier 1 + 2). Competitive query; compete on depth + breadth.
2. **`/guides/find-your-forgotten-pensions`** — the universal "I've worked in multiple countries, what am I owed?" hub. Links to all country-emigration pages + the free mini-tool.
3. **`/guides/retire-abroad-as-an-expat`** — the destination-comparison hub for the 55–62 segment. Links to all retirement-destination pages + the `/simulation` tool.

Each spoke page links back to its pillar. Each spoke page also links to 2–3 neighbouring spokes (e.g. "FR→LU corridor page" links to the "PER deduction" page and the "declaring foreign pension in LU" page). This internal linking is what turns 74 pages into a compounding SEO asset.

## Mini-tools as lead magnets (ship alongside content)

Each doubles as an SEO page AND an email-capture mechanism.

1. **"Check for forgotten Swiss pension" wizard** — highest viral potential; pairs with Trigger-2 pages. MVP: a 4-question form that guides users to zentralstelle.ch + captures email for follow-up.
2. **"Am I using my full PER / 3a / 111bis cap?" calculator** — seasonal traffic driver for Trigger-3 pages. Takes salary + country, returns the unused deduction amount.
3. **"Can I use [pension X] for a property in [country Y]?" decision tree** — Trigger-5 pages. Matrix lookup that's genuinely hard to find elsewhere.
4. **Public version of `/simulation`** — Trigger-4 pages. The existing tool, linkable without signup, saves to email on request.

## What NOT to do

- **Don't chase head terms**. "Pension calculator" / "retirement planning" have CPCs of €8-15 and are dominated by established finance brands. The bootstrap SEO edge is long-tail specificity.
- **Don't ship thin pages**. Google's helpful-content updates penalise thin AI-generated content. Each page needs genuine expert depth — 800-1500 words minimum, with citations to official sources (CNAV, SVA, CNAP, etc.).
- **Don't build before you have validated keywords**. Before Tier 1 goes into production, run each title through a free keyword tool (Google Keyword Planner + Ahrefs free version) to filter out zero-volume queries.
- **Don't skip schema markup**. FAQ + HowTo schema gets you rich snippets, which roughly doubles CTR on ranked pages. Cheap to add; large effect.
